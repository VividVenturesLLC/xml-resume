# 1.2 Document conventions
```
1.2.1 Terminology will be adopted from [HR Open Standards](http://www.hropenstandards.org/),
      HR-XML and HR-JSON standards.

1.2.2 Documention for this project is written in [MD](https://help.github.com/articles/markdown-basics/)
      and Github Flavored Markdown [GFM](https://help.github.com/articles/github-flavored-markdown/).

1.2.3 [UML](http://uml.org/) figures are created with software tool
      [nomnoml](http://www.nomnoml.com/) syntax.
```
